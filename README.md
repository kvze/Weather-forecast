# Weather forecast

Простое веб-приложение для просмотра прогноза погоды в любом городе.

## Возможности

- Поиск погоды по названию города (поддерживаются русские и английские названия)
- Отображение текущей температуры
- Описание погодных условий
- Информация о скорости ветра
- Влажность воздуха
- Атмосферное давление
- Визуальные иконки для различных погодных условий


## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
- OpenWeatherMap API

## Примечание

Для работы приложения необходим API ключ от OpenWeatherMap. По умолчанию при загрузке страницы отображается погода в Москве.
